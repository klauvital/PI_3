O objetivo deste sistema é suprir a necessidade apontada pelo setor imobiliário em relação à precificação de imóveis, que não possui uma maneira fácil, padronizada e acessível que possa ser aplicada.

Estamos oferecendo um sistema web de precificação, que encontre um valor aproximado para determinado imóvel, com base em comparação com outros imóveis com características similares. Para isso, haverá o desenvolvimento de um banco de dados de imóveis da região de Ribeirão Preto, que servirá como ponto de partida para a criação de uma calculadora de precificação, que comparará algumas características específicas utilizadas nas avaliações de imóveis, com os imóveis dessa base de dados desenvolvida.

Para utilização do sistema, favor fazer seu CADASTRO para depois ENTRAR no sistema.

Colabore cadastrando seu imóvel e assim poderemos manter um banco de dados mais robusto e uma avaliação mais precisa.
